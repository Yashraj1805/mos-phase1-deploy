# docs/stories/demo_scope.md

# Phase 1 Demo Scope — MOS

# Goal

Deliver a working multi-agent demo within 2-3 days.

---

# Demo Flow

```text
Create Business Config
        ↓
Submit Lead
        ↓
Analytics AI Scores Lead
        ↓
Sales AI Generates Response
        ↓
Conversation Logged
        ↓
Manager Summary Generated
```

---

# Included Features

## 1. Business Config System

* create config
* store config
* retrieve config

---

## 2. Lead Submission

* manual lead submission
* lead storage
* tenant linkage

---

## 3. Analytics AI

* lead scoring
* intent detection
* lead categorization

---

## 4. Sales AI

* response generation
* qualification logic
* CTA generation

---

## 5. Conversation Logging

* user messages
* AI responses
* intents
* timestamps

---

## 6. Manager Summary

* total leads
* hot leads
* top intents
* recommendations

---

# Excluded Features

## Infrastructure

* Docker

* Kubernetes

## Integrations

* WhatsApp live integration
* CRM sync

* Voice calling

## AI Features

* autonomous learning

# Engineering Priorities

Priority order:

```text
1. Architecture stability
2. Modular structure
3. Working orchestration
4. Agent contracts
5. Demo usability
```

---

# Success Criteria

Demo is successful if:

* business config works
* lead flow works
* orchestrator works
* Analytics AI works
* Sales AI works
* logging works
* summary works

---

# Non-Goals

Phase 1 is NOT focused on:

* production deployment
* scalability
* infra optimization
* enterprise integrations
* advanced DevOps
